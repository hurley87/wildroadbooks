# Sound Effects

This directory contains sound effects for the game. The following files are required:

## Required Sound Files

1. **correct.mp3** - Bright chime/ding for correct answers
2. **wrong.mp3** - Soft buzzer/thud for incorrect answers
3. **partial.mp3** - Neutral tone for partial credit answers
4. **streak.mp3** - Rising whoosh for streak milestones (3x, 5x)
5. **level-up.mp3** - Triumphant jingle for leveling up
6. **milestone.mp3** - Achievement unlock sound for progress milestones
7. **fanfare.mp3** - Victory fanfare for game completion
8. **click.mp3** - Button tap sound

## Free Sound Resources

### Recommended Sources:

1. **Mixkit** (https://mixkit.co/free-sound-effects/game/)
   - High quality, no attribution required
   - Search: "game ui", "quiz", "positive", "success"

2. **Freesound** (https://freesound.org/)
   - Large library, requires account
   - Search: "game positive", "quiz correct", "achievement"
   - Check license requirements

3. **Zapsplat** (https://www.zapsplat.com/)
   - Free with account registration
   - Professional quality game sounds

4. **OpenGameArt** (https://opengameart.org/)
   - UI sound packs available
   - Check individual licenses

## File Format

All files should be in MP3 format for browser compatibility. Recommended settings:
- Format: MP3
- Bitrate: 128 kbps or higher
- Sample Rate: 44.1 kHz
- Duration: 0.5-2 seconds (shorter is better for UI sounds)

## Testing

After adding sound files, test that:
- All sounds play correctly
- Volume levels are balanced
- Sounds don't overlap awkwardly
- Mobile devices handle sounds properly
